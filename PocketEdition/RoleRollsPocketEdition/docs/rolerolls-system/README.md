# RoleRolls System

## Escopo

Esta pasta documenta a regra de negocio do sistema base `RoleRolls`.
O foco aqui e responder como o sistema funciona como RPG: o que compoe uma criatura, como um teste e resolvido, como um ataque vira dano e como o desgaste vira condicao.

Esta fase ainda nao documenta em detalhe o conteudo proprio dos `DefaultUniverses`.

## Regra De Leitura

- `Implementado hoje`: a regra que o sistema realmente aplica hoje.
- `Assumido por testes/balance`: a regra que os testes e os documentos de balance usam como contrato ou modelo.
- `Divergencias e observacoes`: os pontos em que essas duas leituras nao batem perfeitamente.
